◆ yall_enable_debug()

_YALL_PUBLIC yall_error yall_enable_debug ( const char *  subsystem)

Enable the debug mode on the library. This function must be called AFTER yall_init(). Debug message will be wrote using the given subsystem.

Parameters
subsystemName of the subsystem to us as debug subsystem. If the subsystem does not exists, no debug message will be generated. Also, this subsystem should not be modified during debug mode.
Returns
Error code whether it succeed.
