28 /*
29  * This file contain the queue function's prototype. Orignally, this queue was
30  * used to store the log message, while the writer thread was writing them.
31  * It should be lock free to avoid race conditions inside the library. The lock
32  * free property is not hard to implement, as this is not a real queue : the
33  * logger thread add data to the queue, in a stack way : head become head->next
34  * and the new node become the head. The writer thread copy the head pointer and
35  * replace the head value to NULL, which lead to the creation of a "new" queue.
36  */
