◆ swap_queue()

struct qnode* swap_queue ( void  )

As the writer thread get the whole queue when writing, there is no dequeue() function (currently), but a function to return the current's queue head, and the set head to NULL. If the queue is empty, NULL is returned. The system will then add nodes to a new queue.

Returns
The (previously) current queue head.
