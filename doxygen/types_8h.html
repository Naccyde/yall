◆ yall_output_type

enum yall_output_type

This enum stores all the different output types for the subsystems. These values has to be power of 2, as they can be combined to write on multiple outputs.

Enumerator
yall_inherited_output 

Subsystem inherit its output type from its parent.

yall_console_output 

Subsystem log message will be wrote on console.

yall_file_output 

Subsystem log message will be wrote in file.

yall_syslog_output 

Subsystem log message will be redirected to syslog.
