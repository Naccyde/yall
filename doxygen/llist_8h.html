◆ ll_delete()

void ll_delete ( llist_t *  l,
void(*)(void *data)  data_delete 
)

Delete the given linked-list. The data present in each node will be passed to 'data_delete' function pointer. This function can then manage the data or free it ('data_delete' could be a pointer to stdlib's 'free()'). If 'data_delete' is NULL, nothing will be done to the node's data (meaning the data won't be freed). This function doesn't use llist internal parsing function, as we don't need to ensure the list remain consistent between node's freeing.

Parameters
lPointer to the linked-list object.
data_deletePointer to the function managing linked-list node's data.
