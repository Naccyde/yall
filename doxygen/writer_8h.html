yall_error writer_init ( uint16_t  frequency)

Start the writing thread. Also, this function does create a cqueue_t object and send it to the thread. This object is used to share the messages to write between the threads and the writing thread. If the function fail, message queue is deleted. This should not be a problem as no function should use this message queue until writer_init() succeed.

Parameters
frequencyFrequency to use for the writer's thread loops.
Returns
Error code depending of the success or failure of the function.
