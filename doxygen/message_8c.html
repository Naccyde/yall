Create the log message. It fills buffer we given data and specific format. By the way, the function returns the required buffer size of the length parameter is 0, by it does not add room for '
' and '\0'.

Parameters
log_bufferBuffer in which the log message will be wrote.
lenLength of the log buffer.
message_formatFormat to use when generating the log message.
argsArguments to use with the message's format. Can be empty.
Returns
Returns the number of characters wrote. It works the same way as snprintf and friends as if len equals 0, it returns the number of characters than would have been wrote.

Create and returns a new message with the given parameters. This message must be freed with the proper function as internals must be handled specifically. It is useless to test the return value, as it use malloc. If malloc fail, your first problem won't be the NULL returning function.

Parameters
dataData to set inside the message. This value can be NULL.
log_levelLog level of the message.
pParameters used to write the log message. It is used to copy the different configuration parameters.
