◆ cq_swap()

void cq_swap ( cqueue_t *  from,
cqueue_t *  to 
)

Swap the given queue. Nodes from 'from' queue will be set as node from 'to' queue. 'to' queue will then be reversed, meaning the user, once 'cq_swap' called, can call 'cq_dequeue()' on 'to' queue. 'from' queue is then empty. This function is thread safe.

Parameters
fromQueue to swap. After function completion, node list of this queue will be empty. Can't be NULL.
toQueue containing the reversed node once function completes.
Returns
Swapped queue. This queue has to be freed by the user once done. Can't be NULL.
